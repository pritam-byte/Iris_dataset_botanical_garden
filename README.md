# Iris_dataset_botanical_garden

🌸 Iris Flower Classification using Machine Learning
📌 Project Overview

This project focuses on building a Machine Learning classification system to automatically identify species of Iris flowers based on their physical measurements. The goal is to assist botanical researchers in automating plant identification, reducing manual effort and improving accuracy.

🎯 Problem Statement

Botanists traditionally identify Iris flower species manually by measuring sepal and petal dimensions. This process is:

Time-consuming

Error-prone

Difficult to scale for large datasets

To solve this problem, a Machine Learning classification model is developed to automatically predict the species of an Iris flower based on its physical features.

📊 Dataset Description
Feature	Description
SepalLengthCm	Length of sepal (cm)
SepalWidthCm	Width of sepal (cm)
PetalLengthCm	Length of petal (cm)
PetalWidthCm	Width of petal (cm)
Species	Target variable (Iris-setosa, Iris-versicolor, Iris-virginica)
⚙️ Algorithms Used

Three classification algorithms were implemented and compared:

K-Nearest Neighbours (KNN)

Logistic Regression

Naive Bayes
